# 明治大学元父母の会 学生応援イベント一覧

体育会・文化系サークルの試合・公演・展示などの日程データです。
明治大学元父母の会が学生の活動を応援するために収集しています。

## ファイル

- `data/events.csv` — 現在のイベント一覧(2026年6月〜)
- `data/events_archive.csv` — 過去のイベント一覧(2023〜2026年)
- `data/venues.csv` — 会場名・住所マスターデータ
- `data/clubs.csv` — 体育会・公認団体名マスターデータ

## 今後の予定

このリポジトリは、データ収集・整形を段階的に自動化していく予定です。

1. ✅ 手動収集データをこの形でバージョン管理開始
2. 連盟公式サイトからの自動収集(GitHub Actions)
3. AIによる情報の自動整形(Gemini API)
4. Pull Requestでの内容確認フロー
5. GitHub Pages / Googleサイトでの自動公開

## 注意事項

会員個人のお名前・メールアドレス等の非公開情報はこのリポジトリには含めていません。
